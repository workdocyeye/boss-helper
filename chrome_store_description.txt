# 我来直聘 - Chrome扩展详细描述

## 功能介绍

「我来直聘」是一款专为求职者打造的智能Chrome扩展工具，帮助您在BOSS直聘平台上大幅提升沟通效率。

### 主要功能：
• 一键自动打招呼：无需手动点击，批量向心仪的职位发送打招呼消息
• 智能进度跟踪：清晰展示打招呼进度，让您掌握求职状态
• 简洁直观界面：操作简单，一目了然
• 安全可靠：完全在本地运行，不收集用户个人信息

### 为什么选择我们？
✓ 节省宝贵时间：从手动点击到自动化操作，释放您的时间和精力
✓ 提高求职效率：在竞争激烈的求职市场中脱颖而出，抢占先机
✓ 轻量级设计：占用极少系统资源，不影响浏览体验
✓ 持续更新优化：根据用户反馈不断完善功能

## 使用说明

1. 安装扩展后，在BOSS直聘职位列表页面点击扩展图标
2. 点击"开始自动打招呼"按钮
3. 扩展将自动为您浏览职位并发送打招呼消息
4. 随时可以通过"停止自动打招呼"按钮暂停操作

## 隐私保障

我们高度重视用户隐私，「我来直聘」扩展：
• 不收集任何用户个人信息
• 不记录您的账号密码
• 不上传任何数据到外部服务器
• 所有操作均在本地浏览器中完成
